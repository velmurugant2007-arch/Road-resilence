# Experiments

## Directory: `experiments/`

Stores experiment logs, results, and artifacts. Every experiment uses the [Experiment Template](../templates/experiment_template.md) and is logged in the [Research Notebook](../docs/research_notebook.md).

## Naming Convention

```
experiments/
├── EXP-0001-unet-baseline/
│   ├── config.yaml
│   ├── results.md
│   ├── metrics/
│   └── visualizations/
```
