# GIS Processing

## Module: `gis/`

**Owner**: GIS Engineer  
**Status**: Awaiting Phase 7 (Implementation)

## Purpose

Handle all geospatial data operations — CRS transformations, image tiling, radiometric normalization, vectorization, and georeferenced output generation.

## Dependencies

- Receives: Raw satellite imagery
- Produces: Normalized tiles (for AI), georeferenced vector outputs
- Consumed by: AI module, Graph module, Dashboard
