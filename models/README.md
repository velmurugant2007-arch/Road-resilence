# Trained Models

## Directory: `models/`

Stores trained model checkpoints, ONNX exports, and model metadata. Large binary files should use Git LFS.

## Naming Convention

```
models/
├── <model_name>_<dataset>_<date>_<metric>.pth
├── <model_name>_<dataset>_<date>.onnx
└── <model_name>_metadata.json
```
